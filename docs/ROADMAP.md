# Siftwise Roadmap

## 0.x Series (Alpha)

- [x] Core analyzer / planner / executor loop
- [x] Residual concept (flag but do not move)
- [x] Basic CLI commands (draft / review / execute / refine)
- [ ] Better summaries and progress reporting
- [ ] More robust undo journal
- [ ] Confidence threshold tuning and presets

## 1.x Series (Stable Core)

- Cohesion-aware folder moves (keep related trees together)
- Configurable profiles for different archive styles
- Rule learning from user edits and overrides
- Improved test coverage and CI

## 2.x and Beyond (AI-Enhanced)

- Embedding-based similarity for more intelligent grouping
- Optional LLM integration for classification and explanations
- Interactive "focused residual cleanup" workflows
- Cloud / multi-user support and UI wrappers
